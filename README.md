# 📊Etude sur l'eau potable
## 🎯Objectif
**DWFA** (Drinking Water For All) est ONG dont le but est d’améliorer l’accès à l’eau potable dans le monde
La DWFA est présente dans 3 domaines d'expertise
- Création de services d’accès à l’eau potable
- Modernisation de services d’accès à l’eau déjà existants
- Consulting auprès d’administrations / gouvernements à propos des politiques d’accès à l’eau
  
Le but de ce projet est de permettre à un bailleur de fond d’investir dans un pays dans un de ces 3 domaines.  
Pour faire ce choix 5 fichiers ont été fournis:
- **Population** qui contient les populations par année et par pays
- **Région** qui contient la région du monde à laquelle appartient la population
- **Instabilité** politique qui contient l'indice d'instabilité politique par pays et par année
- **Mortalité** qui contient le taux de mortalité par pays pour l'année 2016
- **Accès eau potable** qui contient la taux d'accès à l'eau potable par pays et par année
  
## 🛠️Outils
- **Tableau Desktop** : Visualisation des données
- **Tableau Prep** : Nettoyage des données
